# space-prediction-market-devnet
Utilities, helper scripts and testing notes for the Space prediction market running on Solana Devnet.

About Space

Space is a Solana-based prediction market platform that supports spot and leveraged positions across user-created markets.

This repository is used for internal development and Devnet testing only.

Purpose of this repository
Devnet wallet and faucet testing
Market creation and settlement testing
Claim and portfolio validation
Helper scripts for order book seeding and test scenarios
Notes for QA and bug reproduction
Current test areas
Spot market claims
Leveraged position settlement
Portfolio profit / loss calculations
Open order vs filled order behaviour
Closed market and resolved position states
Devnet SOL top up and wallet funding
Notes

This repository does not contain production code.

All testing is performed against Solana Devnet and uses test wallets, test USDC and test SOL only.
